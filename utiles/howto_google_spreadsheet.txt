You can copy the cell values and when pasting:
Right-click
Paste Special...
Paste values only

------

[ Copy values ]
1. In cell C1, press Ctrl+C to copy (shortcut key for Windows).
2. In Cell D1, press Ctrl+Shift+V or right click on cell D1, select Paste special and apply paste values only.

[ Copy Formula ]
1. Ctrl+C to copy.
2. Alt+E+S+F to paste formulas only.

Open a document, spreadsheet, presentation, or drawing.
Click the File menu and select See revision history.
Click a time stamp in the panel on the right to see a previous version of the file. You’ll also see the people who edited the file below the time stamp, and the edits that each person made is shown in the color that appears next to their name.
If you want to revert to the version you're currently viewing, click Restore this revision.
If you'd like to return to the current version of your document to continue editing, click the X in the upper right of the 'Revision history' panel.


Öffnen Sie ein Dokument, eine Tabelle oder eine Präsentation auf dem Computer.
Klicken Sie oben auf Datei  Versionsverlauf  Versionsverlauf ansehen.
Wählen Sie im rechten Steuerfeld eine frühere Version aus. Sie können sehen, wer die Datei bearbeitet hat. Die Änderungen sind in der Farbe hervorgehoben, die neben dem Namen angezeigt wird.
Optional: Klicken Sie auf das Dreipunkt-Menü   Diese Version benennen, um der Version einen anderen Namen zu geben.
Pro Zeichnung, Dokument oder Präsentation sind 40 benannte Versionen möglich.
Pro Tabelle sind 15 benannte Versionen möglich.
(Optional): Lassen Sie sich Details zu gruppierten Versionen anzeigen:
Wählen Sie im rechten Steuerfeld eine Version aus und klicken Sie dann auf "Detaillierte Version anzeigen" .

Benachrichtigungen einrichten
Sie können nur für sich selbst Benachrichtigungen einrichten. Dabei werden Sie nicht bei Änderungen benachrichtigt, die Sie selbst an Ihrer Tabelle vorgenommen haben, sondern nur bei Änderungen durch andere Nutzer.
Öffnen Sie auf dem Computer eine Tabelle in Google Tabellen.
Klicken Sie oben auf Tools und dann Benachrichtigungsregeln.
Wählen Sie im eingeblendeten Fenster aus, in welchen Fällen Sie Benachrichtigungen erhalten möchten. Sie können Benachrichtigungen erhalten, wenn:
Änderungen vorgenommen wurden: Benachrichtigungen werden gesendet, wenn jemand eine Änderung in einer Tabelle vornimmt.
ein Nutzer ein Formular einreicht: Benachrichtigungen werden gesendet, wenn jemand ein Formular ausfüllt.
Wählen Sie im daraufhin eingeblendeten Fenster aus, "wie oft" Sie Benachrichtigungen erhalten möchten. Erhalten Sie Benachrichtigungen per:
E-Mail als tägliche Übersicht: Es wird täglich eine Zusammenfassung aller Änderungen gesendet.
E-Mail als direkte Info: Es wird bei jeder Änderung direkt eine E-Mail gesendet.
Klicken Sie auf Speichern.
